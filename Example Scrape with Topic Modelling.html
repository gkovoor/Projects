import random
import requests
import sys
import time
import pandas as pd
import wikipedia

data = []
def scrapeArticles(name, count):
    if count == 0:
        sys.exit("Max articles reached")
        
    page = wikipedia.page(name)
    title = page.title
    print("Article No. {} is {}".format(11 - count, title))
    count -= 1

    all_links = {}
    all_links = page.links
    random.shuffle(all_links)

    # Creates a list of lists containing our data
    data.append([page.title, page.content, page.url])
    
    # Time between recursive calls to prevent overloading the Wikipedia server with requests
    time.sleep(1)
    scrapeArticles(random.choice(all_links), count)

The above script works by using the Wikipedia Library to make a set of links on the original page that is searched. It then randomly picks a link to navigate to, and repeats this process until we reach the maximum number of articles.

import nltk
from nltk import FreqDist
import matplotlib.pyplot as plt
import seaborn as sns

# function to plot most frequent terms
def freq_words(x, terms = 30):
  all_words = ' '.join([text for text in x])
  all_words = all_words.split()

  fdist = FreqDist(all_words)
  words_df = pd.DataFrame({'word':list(fdist.keys()), 'count':list(fdist.values())})

  # selecting top 20 most frequent words
  d = words_df.nlargest(columns="count", n = terms) 
  plt.figure(figsize=(20,5))
  ax = sns.barplot(data=d, x= "word", y = "count")
  ax.set(ylabel = 'Count')
  plt.show()

import string
string.punctuation 

def remove_punctuation(text):
    no_punct = []
    for words in text:
        if words not in string.punctuation:
            no_punct.append(words)
    words_no_punct = ''.join(no_punct)
    return words_no_punct

This is a function that appends words to a list thus removing any punctuation. We now pass our Pandas Series through this function. The apply function does this for us.

    lemmatized = []

Building a LDA Model¶

The two main articles I used to understand how gensim can help build LDA models were Topic Modeling with Gensim (Python) and Gensim Tutorial A Complete Beginners Guide. Both of these included code on how to create the bag of words corpus, dictionary and visualisations.

In [140]:

import pyLDAvis
import pyLDAvis.gensim
In [141]:
common_words = pd.Series(new_data['Lemmatized Words']).apply(lambda x: x.split())

To create an LDA model we need two inputs, a dictionary object and a bag-of-words (Corpus).

  • A dictionary in Gensim represents a map of all words (tokens) to its unique id.
  • Using the dictionary we can create a bag-of-words corpus in the form of (word_id, word_frequency).
In [142]:
# Create our Dictionary
dictionary = corpora.Dictionary(common_words)

# Create our Bag of Words Corpus
corpus = [dictionary.doc2bow(words) for words in common_words]

We can use this two inputs to train our LDA model. There are some other parameters we can update:

  • num_topics (The number of topics we want to output)
  • chunksize (The number of documents to be used in each training chunk)
  • passes (The number of training passes)

These can also be found in the Gensim documentation.

In [143]:
# Creating our LDA Model
LDA = gensim.models.ldamodel.LdaModel
lda_model = LDA(corpus=corpus, id2word=dictionary, num_topics=5, random_state=100, passes=50, per_word_topics=True)
